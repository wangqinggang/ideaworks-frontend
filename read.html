为所有的页面元素添加 :hover 属性


FireFox 下，所有的页面元素都有 :hover 属性，当鼠标悬停时，可以变换样式，能给用户一个好的体验。但是 IE 下面，只有超级链接标签（A）才具备此属性，本文介绍的是通过 expression 设置，解决 IE 的这一问题。

虽然 CSS3 还没有完全定稿，但目前已知的一些功能足以让所有的前端开发攻城师激动不已，灵活的背景图片设置、轻松实现的阴影及边框，还有之前只能靠 Javascript 才能实现的动画效果，在 CSS3 中都能由前端攻城师自己来定义。
